About El Polo Loco

What is El Polo Loco?

El Polo Loco is an action-packed 2D browser-based game where you take control of Pepe, a cunning character on a mission to defeat enemies, collect treasures, and escape the clutches of the dreaded boss, Endboss. Dodge obstacles, throw bottles at enemies, and master your agility to complete the level!

Game Features

  • Dynamic Gameplay: Fight enemies, collect coins and bottles, and manage your health
  • Engaging Animations: Smooth character movements and interactive visual effects
  • Boss Battle: Face off against the challenging Endboss with strategic gameplay
  • Audio Experience: Immersive background music and sound effects
  • Multiple Screens: Start screen, game instructions, statistics, and end screens
  • Responsive Design: Play on desktop and mobile devices

Development

El Polo Loco was developed as a comprehensive educational project within the Developer Akademie GmbH training program. It showcases modern web game development techniques using vanilla JavaScript, HTML5, and CSS3.

Developer: Waldemar Chorow
Game Assets & Design: Developer Akademie GmbH
Implementation & Logic: Waldemar Chorow

Technologies Used

This game is built with pure web technologies:

  • HTML5 – Game structure and canvas rendering
  • CSS3 – Styling and responsive design
  • JavaScript (ES6+) – Game logic, collision detection, and animations
  • Object-Oriented Programming: Class-based architecture for maintainable code

How to Play

Controls:

  • Arrow Keys / A/D: Move left and right
  • Space / W: Jump to avoid obstacles and enemies
  • Click / Throw Button: Throw bottles at enemies

Objective: Navigate through the level, defeat enemies using thrown bottles, collect coins and energy bottles, and ultimately defeat the Endboss to win the game!

Educational Purpose

This project serves as a practical demonstration of game development principles and web technologies. It includes aspects such as:

  • Game loop and update cycles
  • Collision detection algorithms
  • Sprite animation and rendering
  • User input handling
  • State management and game flow
  • Performance optimization
